Why It Matters

3~6개월 걸리던 임상시험 재현을 <20분 workflow로 압축

Traditional RCT는 강력하지만 느리고 비싸며, Real-World Evidence는 현실적이지만 인과추론이 어렵습니다. TEAM 3는 그 사이를 잇는 Target Trial Emulation을 멀티에이전트로 구현합니다.

Our Approach 실사용 데이터 위의 가상 임상시험

TTE를 자동으로 설계·실행해 인과적 근거 생성 속도를 높입니다.

What We Do

전통적 RCT 없이, 기존 임상시험을 실사용 데이터로 가상 재현합니다.

We emulate landmark trials on real-world patient data by translating clinical protocol language into structured cohorts, standardized concepts, causal analysis, and publication-ready evidence.

Traditional RCT

강한 인과추론, 낮은 현장성

무작위 배정과 엄격한 통제로 robust causal evidence를 확보하지만, 비용과 기간, 대표성 한계가 큽니다.

Real-World Evidence

현실 데이터, 복잡한 편향 문제

EMR과 청구 데이터는 풍부하지만, 전문가 수작업 없이 신뢰 가능한 인과추론으로 연결하기 어렵습니다.

TEAM 3 TTE

설계부터 보고까지 단일 파이프라인

실사용 관찰 데이터에서 가상의 무작위 임상시험을 설계·실행해, 임상적으로 해석 가능한 실사용 근거를 생성합니다.

System Overview

각 에이전트가 기능을 분담하고, 단계 간 오류 전파를 구조적으로 차단합니다.

Six specialized agents coordinate through explicit state objects rather than a single oversized prompt, which keeps the workflow reproducible and reviewable.

0. Supervisor 전체 워크플로우 관리 및 무결성 검증

Human-in-the-loop checkpoints와 재실행 피드백을 담당합니다.

1. Trial 자연어 구조화

PICO와 적격 기준을 조건 단위로 해석

Output: criteria_struct
2. Mapping 표준 용어 매핑

UMLS 및 OMOP Concept ID로 확장·정제

Output: concept_sets
3. Extraction 코호트 추출

OMOP CDM SQL을 생성·실행해 군을 구성

Output: cohort_sql
4. Analysis 인과추론 및 통계

PSM, IPTW, survival analysis를 자동 수행

Output: analysis_spec
5. Reporting 최종 리포트 생성

KM, HR, 포레스트와 자동 리포트를 생성

Output: report_assets
Modular by design

설계, 매핑, 추출, 분석, 보고를 분산 위임해 컨텍스트 누적을 줄이고, 복잡한 clinical workflow도 단계별로 검증할 수 있게 만들었습니다.
